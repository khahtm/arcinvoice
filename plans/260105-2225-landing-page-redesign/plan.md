# Arc Invoice Landing Page Redesign

**Date:** 2026-01-05
**Status:** In Progress
**Style Reference:** FlowSuite (SaaS Dashboard Landing)

## Overview

Redesign Arc Invoice landing page using FlowSuite's modern SaaS aesthetic - featuring blue gradient hero, dashboard mockups, and clean card-based layout.

## Design System

| Element | Value |
|---------|-------|
| Primary Blue | `#0066FF` |
| Hero Gradient | `#0066FF` → `#3B82F6` |
| Background | `#F0F7FF` → `#FFFFFF` |
| Text Dark | `#1A1A2E` |
| Text Muted | `#6B7280` |
| Card Shadow | `0 4px 24px rgba(0,0,0,0.08)` |
| Border Radius | `16px` (cards), `9999px` (pills) |
| Font Headline | Outfit, 700 weight |
| Font Body | Inter, 400-500 weight |
